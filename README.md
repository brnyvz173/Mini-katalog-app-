# 📱 Mini Katalog App

Flutter ile geliştirilmiş basit bir katalog uygulamasıdır. Kullanıcılar ürünleri listeleyebilir ve detaylarını görüntüleyebilir.

## 🚀 Özellikler
- Ürün listeleme
- Ürün detay sayfası
- Basit ve kullanıcı dostu arayüz

## 🛠️ Kullanılan Teknolojiler
- Flutter
- Dart
